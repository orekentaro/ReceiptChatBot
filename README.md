# レセプトの月末処理をお助けするLINE Chat Bot 『レセプトおじさん』

## 概要
- 月末処理の際に総患者数とコンタクトレンズ(以下CL)の処方の割合を知りたい。
- CLが40％を超えると保険点数的に良くない（らしい）
- 40％未満になる最大の数値になる様に、総数とCL数を引いて行き結果を返す。

## 使い方
-　半角で、『総数』『スペース』『CL数』を入力し送信。
- 例）　782 342
